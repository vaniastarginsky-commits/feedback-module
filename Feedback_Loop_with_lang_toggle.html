What it is

Sound that feeds on itself

Feedback Loop is a stereo analog effects processor that puts you in control of something usually chaotic — the feedback path. Route a diode clipper and a 24dB SSI2144 filter either in the main signal chain or inside the feedback loop itself.

Insert any external module via Send/Return. Shape the return with HP/LP OTA filters, a phase inverter, and a V2164 crossfader. The result: everything from subtle harmonic saturation to full self-oscillating feedback destruction.

One switch press changes the architecture entirely.

What it does

Drive · Distortion

Diode Clipper

DIP-8 socket — swap any diode pair. THAT2181 VCA controls drive level with CV. V2164 crossfader blends dry and clipped signal.

Filter · 24dB LP

SSI2144 Ladder

Sound Semiconductor's classic ladder filter. V/Oct pitch tracking. Independent Q trimmer. Swappable SSOP-16 daughterboard. Self-oscillation capable.

Routing · Logic

Dual SW Routing

74HC74 T-Flip-Flop. One button press moves Clipper or Filter between main path and feedback loop. CV-triggerable. Power-on reset.

Insert · Send/Return

Loop Insert

Stereo insert point inside the feedback path. Patch external delay, reverb, or pitch modules. V2164 Aux crossfader controls blend depth.

It started as a simple feedback loop

The idea was straightforward: a stereo feedback processor where you could route things into the loop and control what happens inside it. A diode clipper for harmonic saturation. A 24dB SSI2144 filter for tonal shaping.

Then came the question: should the clipper be before or inside the loop? The answer was: both, switchable in real time.

// SW1: Clipper → MAIN path or LOOP path // SW2: SSI2144 → MAIN path or LOOP path // 74HC74 T-FF + 74HC14 debounce + CV trigger

Session 66: ERC reached zero

After 66 design sessions — hundreds of schematic edits, ERC cycles, and specification documents — the schematic hit zero errors. 337 components, stereo throughout, fully documented.

The V2164 crossfaders replaced the original passive-pot topology. True VCA-based dry/wet with CV control for both the feedback path and the clipper. Four 3296W Bourns trimmers for L/R balance calibration.

// BOM: 337 components // ERC: 0 errors · 21 warnings (lib mismatch, irreducible) // Status: ready for PCB layout

Now: PCB layout

The schematic is frozen. PCB layout is next — 22HP board, placement zones defined for each functional block. First Rev.1 PCB order to JLCPCB planned after layout and DRC verification.
