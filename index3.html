The Concept

A loop that feeds on itself

Feedback Loop is a stereo analog effects processor built around a diode clipper, a 24dB/oct SSI2144 ladder filter, and a configurable feedback path. Insert anything into the loop — the module routes, shapes, and returns the signal with precision.

Two routing switches (SW1, SW2) determine whether the Clipper and Filter sit in the main signal path or inside the feedback loop — enabling dramatically different sonic characters from the same hardware.

Capabilities

Features

Drive · Distortion

Diode Clipper

Asymmetric soft-clipping with a full VCA drive stage. Germanium-style warmth or aggressive hard clipping — voltage-controlled via CV input.

Filter · 24dB

SSI2144 Ladder

Classic 24dB/oct lowpass from Sound Semiconductor. V/Oct tracking, independent resonance (Q) control with trimmer, stereo-linked on a swappable daughterboard.

Routing · Architecture

Dual Routing Switches

SW1 and SW2 route the Clipper and Filter independently between the main signal path and the feedback loop. One module, four sonic configurations.

Insert · S/R

Send / Return

Stereo insert loop between the feedback path output and the mix stage. Patch any external module into the feedback chain — delay, reverb, pitch shifting.

Patch Ideas

Use Cases

Feedback Destruction

Both Clipper and Filter in LOOP mode. Crank the feedback level until the loop sustains itself. Use the filter cutoff to sculpt the resonant frequency of the self-oscillating feedback.

SW1=LOOP · SW2=LOOP · Q↑ · Feed D/W→wet

Filtered Drive

Clipper in MAIN path, Filter in MAIN path. Classic serial distortion-into-filter topology. Use V/Oct for pitch-tracked filter sweeps over driven bass or leads.

SW1=MAIN · SW2=MAIN · V/Oct → sequencer

Delay Loop Mangling

Patch a delay or reverb into Send/Return. The wet return feeds back through the Clipper and Filter on each loop iteration. Dial in degrading echoes that distort and filter over time.

Send→Delay In · Delay Out→Return · SW1=LOOP
