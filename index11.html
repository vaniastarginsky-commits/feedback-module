Positioning

Not a single effect. A feedback topology you can play.

OUROBOROS is built around a controllable stereo loop. The crossfader drives signal into recursion, the clipper and SSI2144 can move between the main path and the loop, the send/return opens the circuit to external processors, and post-loop HP/LP plus phase control keep the whole system musical even at unstable settings.

What changed

A more complete performance system

The new concept adds a dedicated delay button, internal tap-tempo delay for unpatched send/return operation, assignable trigger control, MCU-driven button logic and clearer state feedback. It stays analog where it matters and uses digital control only where it expands performance and reliability.

Interactive signal flow

See how the architecture moves

Toggle SW1, SW2 and SW3 to see how the clipper, SSI2144 and macro-route move. Toggle SW4 to show the internal delay fallback when the external send/return is unpatched.

SW1 · clipper position

Moves the diode clipper between the main path and the loop. This changes both tone and how recursion builds.

SW2 · SSI2144 position

Moves the 24dB filter between the main path and the loop. In-loop operation turns it into a behavioural control, not just EQ.

SW3 · macro route

Switches the architecture between loop behaviour and a more linear series-like route. This is the fast topology change.

SW4 · internal delay

Tap tempo on short presses, very long press for delay on/off. When send/return is unpatched, the internal PT2399 delay can occupy that space.

Selling points

Why this module stands out

Routing as sound design

Repositionable character blocks

The clipper and SSI2144 do not simply sit in a fixed chain. They can move between the main path and the loop, so the architecture itself becomes part of the sound.

Playable recursion

Feedback you can control

Crossfade into the loop, shape energy with LP / HP, destabilize with phase, or tame the loop for a focused stereo processor. The whole point is controllable recursion.

External / internal duality

Send/return plus fallback delay

Patch external processors into the stereo loop, or leave send/return open and let the internal tap-tempo delay take over. It works as an insert hub and as a self-contained effect.

Performance control

Assignable trigger intelligence

One trigger input can be reassigned to SW1, SW2, SW3 or SW4. That means one external clock or gate source can become routing, macro-topology or delay tap control.

Quick manual

How to explain it in one minute

1Build the loop

  • Feed signal with FEEDBACK MIX.
  • Use S/R MIX for loop amount and external insert blend.
  • Start subtle, because the module can escalate quickly.

2Move the character

  • SW1 moves the clipper.
  • SW2 moves the SSI2144.
  • SW3 changes the macro-route between loop and series behaviour.

3Stabilize or break it

  • Use LP and HP to control spectral buildup.
  • Use PHASE to create widening, cancellation or unstable reinforcement.
  • FILTER DRIVE and CLIP DRIVE decide how hard the loop bites back.

4Use the internal delay

  • When send/return is unpatched, internal delay can occupy that slot.
  • SW4 short taps set tempo. Two taps already work; later taps refine the average.
  • Very long press toggles delay on/off.

5Assign trigger control

  • Long press SW1, SW2, SW3 or SW4 to assign the global trigger input.
  • Assigned trigger behaves like a virtual short press for that button.
  • If assigned to SW4, the trigger input becomes external tap tempo.

6Choose your panel language

  • The symmetrical panel concept allows a symbol side and a text side.
  • One side can stay occult and compact; the reverse can be explicit and technical.
  • Same mechanics, two reading styles.
